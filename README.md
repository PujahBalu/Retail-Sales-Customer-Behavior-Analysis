# Retail-Sales-Customer-Behavior-Analysis

# Retail Intelligence Platform

An end-to-end retail analytics project that analyzes sales, customer behavior, product performance, promotions, returns, and regional trends.

## Business Objective

The project helps retail decision-makers identify revenue drivers, underperforming products, valuable customer segments, and opportunities to improve sales and retention.

## Tools Used

* Python
* SQL
* Power BI
* Pandas
* Scikit-learn
* Plotly and Matplotlib

## Key Analysis

* Monthly and yearly sales trends
* Product and category performance
* Customer segmentation using RFM analysis
* Customer lifetime value
* Promotion effectiveness
* Return-rate analysis
* Regional and store performance
* Revenue forecasting

## Project Structure

```text
data/          Dataset and cleaned files
notebooks/     Python analysis notebooks
sql/           SQL queries
dashboard/     Power BI dashboard
reports/       Executive summary
images/        Dashboard screenshots
```

## Key Business Insights

* Identified the products and customer segments contributing the most revenue.
* Evaluated the impact of discounts and promotions on profitability.
* Detected underperforming categories, stores, and regions.
* Developed recommendations for customer retention and inventory planning.
